# task-5

## Описание

Каталог клиентов с возможностью добавления, поиска и фильтрации

### Запуск
* Установка:

    `$ python -m venv .venv`

    `$ source .venv/bin/activate`

    `$ pip instal --requirement requirements.txt`

* Запуск сервера:
 
    `FLASK_ENV=development FLASK_APP=wsgi flask run`

* Запуск тестов

    `python -m pytest tests/`
